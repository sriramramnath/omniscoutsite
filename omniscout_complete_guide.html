OmniScout — anyone can set this up

Run 3 commands once. Then just ask your AI in plain English — it will browse, search, and fill in forms for you automatically.

Part 1 — one time only

Open a terminal on your computer

1

Switch on your laptop and wait for it to fully load

Make sure you can see your desktop before continuing.

2

Open the terminal (the text window where you type commands)

This is where you will type the 3 install commands below. Here is how to open it on each system:

Mac

Press Command ⌘ + Space — a search bar appears. Type Terminal and press Enter.

Windows

Press the Windows key on your keyboard. Type cmd and press Enter. A black window opens — that is the Command Prompt.

Linux

Press Ctrl + Alt + T — the terminal opens immediately.

Part 2 — run these 3 commands

Copy each line below, paste it into the terminal, press Enter, and wait for it to finish before typing the next one

Each command installs one thing. The whole process takes about 2–5 minutes depending on your internet speed.

The 3 commands — type these in order

1
pip install omniscout
2
omniscout install
3
omniscout install --skill
3

Command 1 — installs OmniScout on your computer

pip install omniscout

If you see a message saying "pip is not recognised", you need to install Python first. Go to python.org, download the installer, run it, then try again.

4

Command 2 — picks a browser and downloads the AI model

omniscout install

It will ask you which browser to use (Chrome, Edge, Brave, etc.). Press Enter to accept the default. It then downloads a small AI model — this takes 1–3 minutes.

If you do not have any Chromium browser, add --bundled to the end: omniscout install --bundled

5

Command 3 — tells your AI apps that OmniScout exists

omniscout install --skill

This is the most important step. It writes instructions into Claude Code, Codex, Kiro, Cursor, and others so that whenever you ask them to search, browse, or fill in a form, they automatically use OmniScout without you having to explain anything.

Part 3 — you are done. forever.

Every day from now on — just talk to your AI normally

✓

Open any AI app — Claude Code, Codex, Kiro, Cursor, or any other

Type /omniscout at the start of your message to activate it, then just describe what you want in plain English. The AI handles everything from there.

/omniscout find me the best price for AirPods Pro in India /omniscout go to my Gmail and find any unread emails from last week /omniscout search for Python tutorials for beginners and summarise the top 3
